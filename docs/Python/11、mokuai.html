# 11 模块

# 什么是模块

模块是包含一组代码或一组函数的文件，可以包含在一个应用程序中。一个模块可以是包含单个变量、函数或大规模代码库的文件。

# 创建模块

要创建模块，我们在一个 Python 脚本中编写代码并保存为 .py 文件。 在项目文件夹中创建一个名为 mymodule.py 的文件。让我们在此文件中编写一些代码。

# mymodule.py 文件
def generate_full_name(firstname, lastname):
    return firstname + ' ' + lastname

在项目目录中创建 main.py 文件并导入 mymodule.py 文件。

# 导入模块

要导入文件，我们使用 import 关键字和文件名。

# main.py 文件
import mymodule
print(mymodule.generate_full_name('Asabeneh', 'Yetayeh')) # Asabeneh Yetayeh

# 从模块中导入函数

我们可以在一个文件中有很多函数，并且我们可以分别导入所有函数。

# main.py 文件
from mymodule import generate_full_name, sum_two_nums, person, gravity
print(generate_full_name('hua','joffery'))
print(sum_two_nums(1,9))
mass = 100
weight = mass * gravity
print(weight)
print(person['firstname'])

# 从模块中导入函数并重命名

在导入过程中，我们可以重命名模块名称。

# main.py 文件
from mymodule import generate_full_name as fullname, sum_two_nums as total, person as p, gravity as g
print(fullname('hua','joffery'))
print(total(1, 9))
mass = 100
weight = mass * g
print(weight)
print(p)
print(p['firstname'])

# 导入内置模块

像其他编程语言一样，我们也可以通过使用关键字 import 导入文件/函数来导入模块。让我们导入我们大多数时候会使用的常见模块。一些常见的内置模块：math、datetime、os、sys、random、statistics、collections、json、re

# OS 模块

使用 Python os 模块可以自动执行许多操作系统任务。Python 中的 OS 模块提供了创建、更改当前工作目录和删除目录（文件夹）、获取其内容、更改和识别当前目录的函数。

# 导入模块
import os
# 创建目录
os.mkdir('directory_name')
# 更改当前目录
os.chdir('path')
# 获取当前工作目录
os.getcwd()
# 删除目录
os.rmdir()

# Sys 模块

sys 模块提供用于操作 Python 运行时环境不同部分的函数和变量。函数 sys.argv 返回传递给 Python 脚本的命令行参数列表。此列表中索引 0 处的项始终是脚本的名称，索引 1 处是从命令行传递的参数。 示例 script.py 文件：

import sys
#print(sys.argv[0], argv[1],sys.argv[2])  # 这一行会打印出来：文件名 argument1 argument2
print('Welcome {}. Enjoy  {} Language!'.format(sys.argv[1], sys.argv[2]))

现在要查看这个脚本的工作效果，我在命令行中写：

python script.py joffery Python

结果：

Welcome joffery. Enjoy  Python Language!

一些有用的 sys 命令：

# 退出 sys
sys.exit()
# 知道最大整数变量
sys.maxsize
# 知道环境路径
sys.path
# 知道使用的 Python 版本
sys.version

# 统计模块

统计模块提供用于数值数据的数学统计的函数。这个模块中定义的流行统计函数：mean、median、mode、stdev 等。

from statistics import * # 导入所有统计模块
ages = [20, 20, 4, 24, 25, 22, 26, 20, 23, 22, 26]
print(mean(ages))       # 平均值 ~22.9
print(median(ages))     # 中位数 23 (将数据按从小到大排序后，位于中间位置的那个数。)
print(mode(ages))       # 众数 20 (数据集中出现次数最多的数值)
print(stdev(ages))      # 标准差 ~2.3 (标准差越大，表示数据波动越大；标准差越小，表示数据越集中)

# 数学模块

包含许多数学运算和常数的模块。

import math
print(math.pi)           # 3.141592653589793, pi 常数
print(math.sqrt(2))      # 1.4142135623730951, 平方根
print(math.pow(2, 3))    # 8.0, 指数函数
print(math.floor(9.81))  # 9, 向下取整
print(math.ceil(9.81))   # 10, 向上取整
print(math.log10(100))   # 2, 以 10 为底的对数

现在，我们已经导入了包含大量函数的 math 模块，可以帮助我们进行数学计算。要检查模块中包含哪些函数，我们可以使用 help(math) 或 dir(math)。这将显示模块中的可用函数。如果我们只想从模块中导入特定的函数，我们可以这样导入：

from math import pi
print(pi)

也可以一次导入多个函数

from math import pi, sqrt, pow, floor, ceil, log10
print(pi)                 # 3.141592653589793
print(sqrt(2))            # 1.4142135623730951
print(pow(2, 3))          # 8.0
print(floor(9.81))        # 9
print(ceil(9.81))         # 10
print(math.log10(100))    # 2

但如果我们想要导入数学模块中的所有函数，可以使用 “ * ” .

from math import *
print(pi)                  # 3.141592653589793, pi 常数
print(sqrt(2))             # 1.4142135623730951, 平方根
print(pow(2, 3))           # 8.0, 指数
print(floor(9.81))         # 9, 向下取整
print(ceil(9.81))          # 10, 向上取整
print(math.log10(100))     # 2

当我们导入时，我们也可以重命名函数名称。

from math import pi as  PI
print(PI) # 3.141592653589793

# 字符串模块

字符串模块是一个很有用的模块。下面的例子展示了字符串模块的一些用法。

import string
print(string.ascii_letters) # abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ
print(string.digits)        # 0123456789
print(string.punctuation)   # !"#$%&'()*+,-./:;<=>?@[\]^_`{|}~

# 随机模块

现在你已经熟悉了导入模块。让我们再进行一次导入以熟悉它。让我们导入 random 模块，它会给我们一个 0 到 0.9999 之间的随机数。random 模块有很多函数，但在本节中，我们将只使用 random 和 randint。

from random import random, randint
print(random())   # 它不需要任何参数；它返回一个介于 0 和 0.9999 之间的值
print(randint(5, 20)) # 它返回一个介于 [5, 20] 之间的随机整数（含边界）

# 总结

# 1. 模块的基础操作

  • 本质：一个以 .py 结尾的 Python 文件就是一个模块。
  • 导入方式：
    • import mymodule: 导入整个模块，使用时需加前缀 mymodule.func()。
    • from mymodule import func: 直接导入特定函数，使用时无需前缀。
    • 重命名 (as): 为了简化调用或避免命名冲突，可以给模块或函数起别名（如 import pandas as pd）。
    • 全导入 (\*): 导入模块内所有内容，但需注意可能引发的命名空间污染。

# 2. 强大的内置模块

文档重点列举了 Python 标准库中几个极其常用的模块：

模块名核心功能典型应用
os操作系统接口创建文件夹 (mkdir)、获取路径 (getcwd)。
sys系统相关参数处理命令行参数 (sys.argv)、查看环境路径。
math数学运算处理 pi、开平方 (sqrt)、取整 (floor/ceil)。
random随机数生成生成 0-1 随机浮点数、生成指定范围整数 (randint)。
statistics统计分析计算平均值 (mean)、中位数、标准差。
string字符串常量快速获取字母表、数字集或标点符号集。

# 3. 命令行交互 (sys.argv)

# 文档展示了如何让 Python 脚本接收外部参数。通过 sys.argv，你可以从终端（Terminal/CMD）向程序传递信息，这使得脚本具有更强的通用性和自动化能力。

# 4. 探索模块的技巧

当面对一个陌生的模块时，文档提到了两个非常有用的内置函数：

  • dir(module): 列出模块中定义的所有标识符（变量、函数、类）。
  • help(module): 查看模块的详细帮助文档和功能说明。
